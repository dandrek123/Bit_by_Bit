# ScamSafe - Scam Reporting Community Forum

ScamSafe is a full-stack MERN web application developed as a Florida State University Software Engineering Capstone project.

## Features

- Report online scams
- Vote on scam reports
- Comment and discuss incidents
- User profiles
- Search scam reports
- Community-driven scam awareness

## Technologies

- MongoDB
- Express.js
- React
- Node.js
- JavaScript
- HTML
- CSS

## My Contributions

- Frontend development
- Home page enhancements
- About page development
- FAQ implementation
- UI improvements
- Testing and debugging
- Team collaboration using Git and GitHub

## Project Purpose

The platform helps users share scam experiences, identify suspicious activity, and improve community awareness of online fraud.
